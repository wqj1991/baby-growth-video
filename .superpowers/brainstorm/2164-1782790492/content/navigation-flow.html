🔀 核心交互路径

用户典型操作流
新建项目 → 选择宝宝 → 项目信息 → 生成周期 → 扫描照片 → 项目概览
选照片 → 选择周期 → 扫描文件夹 → 加入待选区 → 设为最终 → 下一周期...
生成视频 → 配置参数 → 选择输出路径 → 后端生成 (进度条) → 查看/下载
