# 周期选择最终照片流程设计

│                │  ┌─────────────────────────────────┐  │
│  ┌──────────┐  │  │  标签页: 全部照片  |  视频       │  │

## 二、各区域交互细节

### 2.1 「全部照片」标签页 - 照片卡片

```
默认状态:
┌──────────┐
│          │
│  照片    │
│  缩略图  │
│          │
│   [小标记]│  ← 右下角状态标记（待选区/最终）
└──────────┘

悬停状态:
┌──────────────────┐
│  [加入待选区]     │  ← 顶部操作按钮栏
│  [设为最终]       │
├──────────────────┤
│                  │
│    照片缩略图     │
│                  │
│     [小标记]      │  ← 右下角状态标记
└──────────────────┘

双击 → 打开大图预览
```

**状态标记（右下角）:**
- 🟡 黄色点：已加入待选区
- ✅ 绿色勾选：已设为最终照片

**悬停按钮逻辑:**
- 未加入待选区 → 显示「加入待选区」按钮
- 已加入待选区 → 显示「从待选区取消」按钮
- 未设为最终 → 显示「设为最终」按钮
- 已设为最终 → 显示「取消最终」按钮

### 2.2 「待选区」面板 - 照片卡片

```
┌────────────────────┐
│                    │
│   照片大图对比     │
│                    │
│                    │
├────────────────────┤
│  [X] 文件名         │  ← 底部：移除按钮 + 文件名
└────────────────────┘

单击照片 → 切换勾选状态（用于拼图）
悬停 → 显示「设为最终」按钮（渐变遮罩层）
双击 → 打开大图预览
```

**交互逻辑:**
- **单击照片卡片**: 切换勾选状态（勾选/取消勾选），用于拼图
- **悬停**: 底部显示渐变遮罩层，内含「设为最终」按钮
- **点击X按钮**: 从待选区移除
- **双击**: 打开大图预览

### 2.3 「视频帧查看器」- 帧卡片

```
┌─────────────┐
│             │
│   帧缩略图   │
│             │
├─────────────┤
│ [☑] [加入]  │  ← 勾选框 + 直接加入按钮
└─────────────┘

双击 → 打开大图预览
```

**交互逻辑:**
- **勾选框**: 选择多个帧，用于批量操作
- **直接加入按钮**: 单独将该帧加入待选区
- **底部确认按钮**: 将所有勾选的帧批量加入待选区
- **双击**: 打开大图预览

## 三、完整流程

### 3.1 流程A：单张照片选定

```
1. 用户在周期进度条选择一个周期
2. 左侧待选区加载该周期已标记为候选的照片
3. 用户在右侧"全部照片"中浏览照片
4. 悬停照片，点击「加入待选区」→ 照片加入待选区
5. 在待选区中悬停某张照片，点击「设为最终」→ 设为最终照片
   OR 在全部照片中悬停，点击「设为最终」→ 直接设为最终
6. 周期状态更新为"已确认"，进度条显示绿色勾选
7. 全部照片列表中该照片显示边框+勾选标记
```

### 3.2 流程B：多选拼图（一次性操作）

```
1. 用户在周期进度条选择一个周期
2. 左侧待选区加载该周期已标记为候选的照片
3. 用户在右侧浏览照片，悬停点击「加入待选区」
4. 在待选区中单击照片卡片 → 勾选用于拼图（可多选2-4张）
5. 点击"生成拼图"按钮
6. 弹出模板选择器，选择拼图模板
7. 进入拼图编辑模式，调整各照片位置/旋转
8. 点击"生成"，拼图作为新照片添加到待选区
9. 用户可以继续编辑或悬停点击「设为最终」
```

### 3.3 流程C：视频截帧

```
┌─────────────────────────────────────────────────────┐
│  C1: 手动截取单帧                                    │
│  1. 在右侧切换到"视频"标签                            │
│  2. 点击视频打开播放器                                │
│  3. 播放/暂停找到合适画面，点击截图                    │
│  4. 截图自动加入待选区                                │
│  5. 后续流程同流程A或流程B                            │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│  C2: 自动生成多帧                                    │
│  1. 在右侧切换到"视频"标签                            │
│  2. 点击视频的"截帧"按钮                              │
│  3. 设置截帧参数（数量或间隔）                         │
│  4. 自动生成视频帧，打开帧查看器                        │
│  5. 用户在帧查看器中勾选或点击「直接加入」              │
│  6. 选中的帧加入待选区                                │
│  7. 后续流程同流程A或流程B                            │
└─────────────────────────────────────────────────────┘
```

## 四、状态同步规则

| 操作 | 待选区 | 全部照片列表标记 |
|------|--------|-----------------|
| 加入待选区 | ✅ 添加 | 🟡 黄色点 |
| 从待选区移除 | ✅ 移除 | ❌ 移除标记 |
| 勾选用于拼图 | ✅ 标记勾选 | ❌ 不同步（一次性操作） |
| 设为最终 | ✅ 标记最终 | ✅ 边框+勾选 |
| 取消最终 | ✅ 取消标记 | ❌ 移除标记 |

## 五、周期切换行为

- 切换周期时，自动加载当前周期已标记为候选（is_selected=true）的照片
- 待选区内容随周期切换而更新
- 之前周期的候选状态保留在数据库中

## 六、视频帧与照片统一处理

- 视频帧在待选区中与普通照片行为完全一致
- 支持单独选定、用于拼图、移除等所有操作
- 设为最终照片时，同样在视频帧列表中显示边框+勾选标记

## 七、双击预览规则

**所有区域的照片/视频帧**（全部照片列表、待选区、帧查看器）都支持**双击预览大图**。
